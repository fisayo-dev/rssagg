# Contributing

## Prerequisites

1. Install Go.
2. Download the source repo and switch the working directory:

```bash
git clone https://github.com/fisayo-dev/rssagg
cd rssagg
```

## Submitting a Pull Request

A typical workflow is:

1. [Fork the repository](http://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo).
2. [Create a topic branch](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-branches).
3. Implement your new feature, changes or improvements.
4. [Add, commit and push your changes](https://docs.github.com/en).
5. [Submit a pull request.](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests)
